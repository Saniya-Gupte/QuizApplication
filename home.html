Let's see how well you know Data Structures!

ARRAYS

It's a data structure that contains a group of elements

STACKS

Data structure used to store a collection of objects.

LINKED LISTS

It's sequence of data structures, which are connected together via links.

QUEUES

Similar to stack which follows First-In-First-Out methodology

BINARY TREES

It's a non-linear data structure in which each node has maximum of two child nodes

HEAPS

Tree based data structure

HASH MAPS

A HashMap is a data structure that is able to map certain keys to certain values.

GRAPHS

It's a non-linear data structure consisting of nodes and edges.
